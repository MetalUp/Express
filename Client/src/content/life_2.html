Life is normally played on a rectangular grid that wraps the left and right edges, and also the the top and bottom edges. The grid you are using already does the left-right wrapping. Top-bottom wrapping means that asking for:

  • cell 400 returns the value of cell 0
  • cell 401 returns the value of cell 1
  • cell -1 returns the value of cell 399
  • cell -2 returns the value of cell 398, and so on...

To achieve this we need a function named KeepWithinBoundskeep_within_bounds that takes in an integer value named i and returns an integer value that will always be in the range 0-399 as shown above. It should work correctly for input values in the range -21 to +420.

Test your function manually, in the Expression pane, then run the automated tests.

You can then apply this function to retrieving values, for example:

NeighbourCells[KeepWithinBounds(403)] neighbour_cells[keep_within_bounds(403)]

and checking that all the values in the returned list are in the range 0-399 and correct.
