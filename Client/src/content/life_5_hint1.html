def will_live(alive, neighbours) : return expression static bool WillLive(bool alive, int neighbours) => expression;

You will need to use one or more logical operators for example:

and or not && || !

You may also use Ternary operator, for example:

result_if_true if condition else result_if_false condition ? result_if_true : result_if_false

You may either implement the three rules using multiple conditional expressions, one nested inside another, or you may choose to use more logical operators to implement the three rules as a single expression without any need the ternary operator. Both approaches are equally valid.
