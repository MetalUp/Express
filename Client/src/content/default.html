In Expression pane on the left, you may enter any expression, simple:

(3+4)/5

or complex:

map(lambda x : x * x, range(1,10)) Enumerable.Range(1,10).Select(x => x * x)

When you hit Enter, the expression will be evaluates and displayed in the Result pane. Don't worry if you have no idea what the second example means for now - try them both now.

Note that:

  • Cursor-up key will retrieve previous expressions that you have entered
  • Cursor-down (from the last expression) will clear the field and enter a new one.
  • If you enter anything that is not a valid expression into this pane, it will be rejected.

Now click in the Functions pane. Here you may enter any 'pure' function: one that simply returns the value of an expression, which will typically make use of values passed into the function as 'parameters', for example:

def square(a) : return a*a static int Square(int a) => a*a;

Type (copy/paste is deliberately not permitted) that function into the Functions pane, and hit Submit. If there are no errors, you will see 'Compiled OK' - if not, correct your code. You can then call use your new function from within the Expression pane, for example with:

square(3) + square(4) Square(3) + Square(4)
