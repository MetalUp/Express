Using Ternary operators your code might look like this:

def will_live(alive, neighbours) : return (variable > const and variable < const) if variable else variable == const

Using only logical operators your code might look like this:

def will_live(alive, neighbours) : return \
(variable and variable > _ and variable < _) \
or (not variable and variable == _)

Note that line continuations - \ - have been added here to break up the single expression the terms more clearly. You may format your code like this, or put it all on a single line, without the added line-breaks, letting it wrap automatically in the pane.
