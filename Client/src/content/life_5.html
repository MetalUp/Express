The rules of, Life state that, for each new 'generation':

  • Any live cell with two or three live neighbours survives.
  • Any dead cell with three live neighbours becomes a live cell.
  • All other live cells die in the next generation. Similarly, all other dead cells stay dead.

Your task is to define a function named WillLivewill_live, that determines whether a single cell will be alive in the next generation.

WillLivewill_live should define two parameters in the order given here:

  1. A Boolean parameter named alive indicating whether the cell itself is currently alive (trueTrue) or not
  2. An integer parameter named neighbours specifying the number of neighbouring cells (from a maximum of 8) that are currently alive.

These two parameters provide sufficient input to implement all the 'generation' rules stated above.

WillLivewill_live should should be implemented as a single expression that embodies all three of the rules above, returning a Boolean value: trueTrue if the cell will be alive in the next generation.
