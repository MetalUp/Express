Using ternary operators the expression returned by your function might look like this:

(variable > const and variable < const) if variable else variable == const (variable ? variable > const && variable < const : variable == const)

Using only logical operators your code might look like this:

(variable and variable > const and variable < const) or (not variable and variable == const) (variable && variable > const && variable < const) | (!variable && variable == const);

You will need to fill in the appropriate variables and constant values
